# 서버 Segmentation 학습 핸드오프 계획

## 목표

서버에서 Codex를 다시 실행했을 때 X-ray 병변 segmentation 모델 학습을 바로 이어갈 수 있도록 현재 상태, 학습 절차, 검증 절차, Git 반영 범위를 문서화한다.

## 범위

- 서버 접속 자체는 수행하지 않는다.
- 백엔드 코드는 수정하지 않는다.
- 실제 데이터나 모델 대용량 산출물은 생성하지 않는다.
- 서버 학습자가 따라 할 수 있는 명령어와 주의사항을 남긴다.

## 작업

1. 현재 구현/커밋 상태 확인
2. 서버 학습 핸드오프 문서 작성
3. worklog/result 작성
4. 문서 파일 검증
